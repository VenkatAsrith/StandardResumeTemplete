# Standard Resume Template

A clean, modern, and responsive HTML-based resume generator designed to easily build, customize, and export professional resumes.

---

##  Live Demo

You can view and test the live application directly in your browser:  
 **[View Live Demo]: https://vinuraresumebuilder.netlify.app/**

---

##  Features

* **Instant Customization:** Simple and easy to modify directly in HTML or form fields.
* **Export to PDF:** Built-in styling optimized for clear, high-quality printing or saving as a PDF (`Ctrl + P` / `Cmd + P`).
* **Responsive Design:** Looks clean and well-structured on desktops, tablets, and mobile devices.
* **Zero Dependencies:** Runs natively in any modern browser without requiring external software or frameworks.
